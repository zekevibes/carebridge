Place your images here and commit/push.

Files expected:
- assets/hero.jpg  — hero/background image (optimized JPG)
- assets/logo.png  — logo image (PNG or SVG preferred)

Recommendation:
- hero.jpg: around 1600×900 or 1920×1080, optimized for web (quality 70-80).
- logo.png: around 400×120 (or SVG) with transparent background.

After adding images, run:

    cd C:\Users\User\carebridge
    git add assets/hero.jpg assets/logo.png
    git commit -m "Add hero and logo images"
    git push origin main

This will let the site show the background and logo in the hero.
